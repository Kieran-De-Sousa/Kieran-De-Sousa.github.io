Project

Blender - Polaroid

One Step+

Blender was my starting point into 3D modelling, with my first project aiming to accurately scale and create the Polaroid One Step+ camera. This project I found very relaxing and fun. With this model, I began picking up ways of how-to 3D model, how to illustrate different shapes and angles of an object and how to a model for use within other projects. The result of this project gave me an understanding of the fundamentals of 3D modelling which I would later use in my Autodesk Maya projects.
